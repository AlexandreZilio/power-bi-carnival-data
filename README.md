# **Power BI Project – Exercises (Lessons 03–05)**

This repository contains a Power BI dashboard developed as part of a training program focused on data visualization and business intelligence using Microsoft Power BI. The project includes practical exercises from lessons 03 to 05, demonstrating essential skills for transforming raw data into actionable insights.

## **Project Overview**

The dashboard applies key Power BI capabilities, including:

- Data extraction, transformation, and modeling using Power Query  
- Creation of interactive visualizations and KPIs  
- Use of relationships between tables to structure a star-schema model  
- Custom measures and calculated fields using DAX  
- Filters, slicers, and drill-down interactions to enhance user experience  

## **Objectives**

- Strengthen Business Intelligence skills through practical exercises  
- Build a clean, dynamic, and decision-oriented dashboard  
- Develop real-world, business-focused analytical capabilities  

## **Tools & Technologies**

- Microsoft Power BI Desktop  
- DAX (Data Analysis Expressions)  
- Power Query (M Language)  
- Relational Data Modeling
